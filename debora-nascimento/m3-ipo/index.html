Este módulo vai cobrir o framework IPO — todo trabalho é uma máquina simples de Input (o que entra), Process (o que acontece) e Output (o que sai). Declarar os três antes de pedir à IA é o que faz a clareza do prompt nascer da clareza do raciocínio.

Vai trazer casos administrativos genéricos (organizar pauta de reunião, redigir comunicado, montar planilha de controle) com IPO declarado explicitamente — pra você adaptar para qualquer tarefa.
